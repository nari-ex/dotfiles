# Dotfile and iTerm configration files

### Create Symlinks

```
cd /path/to/workdir
ln -sf $PWD/git/.gitconfig ~/
ln -sf $PWD/git/.gitignore ~/
ln -sf $PWD/zsh/.dir_colors ~/
ln -sf $PWD/zsh/.zshrc ~/
ln -sf $PWD/direnv/.bin ~/
ln -sf $PWD/tmux/.tmux.conf ~/
ln -sf $PWD/vim/.vimrc ~/
ln -sf $PWD/vim/.vim ~/
```
